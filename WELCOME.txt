🎯 COMMON TASKS:

   Change storage location
   → Edit backend/config.py → STORAGE_PATH

   Change watch folder
   → Edit client/config.py → WATCH_FOLDER

   Enable security
   → Edit backend/config.py → REQUIRE_AUTH = True

   Fix issues
   → Run test-setup.py
   → Read TROUBLESHOOTING.md
