📋 DEPLOYMENT STEPS:

STEP 1️⃣: Create GitHub Repository
──────────────────────────────────

1. Open: https://github.com/new
2. Repository name: personal-cloud-storage
3. Public or Private: YOUR CHOICE
4. ⚠️ DO NOT check "Initialize with README"
5. Click "Create repository"

═══════════════════════════════════════════════════════════════════

STEP 2️⃣: Push to GitHub  
──────────────────────────────────

Run these commands:

git remote add origin https://github.com/YOUR_USERNAME/personal-cloud-storage.git
git push -u origin main

(Replace YOUR_USERNAME with your actual GitHub username)

═══════════════════════════════════════════════════════════════════

STEP 3️⃣: Deploy Frontend to Vercel
──────────────────────────────────

cd frontend
vercel login
vercel --prod

Follow the prompts:
- Set up and deploy? YES
- Which scope? YOUR ACCOUNT
- Link to existing project? NO
- What's your project's name? personal-cloud-storage
- In which directory is your code located? ./
- Want to override the settings? NO

═══════════════════════════════════════════════════════════════════

STEP 4️⃣: Setup Backend (Your PC)
──────────────────────────────────

Terminal 1:
cd backend
python app.py

Terminal 2:
ngrok http 5000

Copy the ngrok URL: https://abc123.ngrok-free.app

═══════════════════════════════════════════════════════════════════

STEP 5️⃣: Configure Vercel Environment Variable
──────────────────────────────────

1. Go to: https://vercel.com/dashboard
2. Select your project: personal-cloud-storage
3. Settings → Environment Variables
4. Add New:
   - Name: REACT_APP_API_URL
   - Value: https://YOUR-NGROK-URL.ngrok-free.app
   - Environment: Production
5. Click "Save"

═══════════════════════════════════════════════════════════════════

STEP 6️⃣: Redeploy with Environment Variable
──────────────────────────────────

cd frontend
vercel --prod

🆘 TROUBLESHOOTING:

Frontend can't reach backend:
→ Check ngrok is running
→ Verify REACT_APP_API_URL in Vercel settings
→ Check backend CORS settings

Vercel deployment fails:
→ Make sure you're in frontend/ directory
→ Run: npm install
→ Try: vercel --prod --force

GitHub push fails:
→ Configure git credentials
→ Use: gh auth login (GitHub CLI)
